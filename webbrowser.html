Web browsers

A client side software application used to locate, retrieve and display content on the world wide web.

The web browser sends Hyper text protocol(HTTP) requests and renders Hyper text mark up language(HTML).

  • Retrieves web pages from servers
  • Displays content using HTML and CSS
  • Supports multimedia content
  • Allows navigation through hyperlinks

Examples of web browsers are Google chrome, Mozilla firefox, Microsoft Edge and Opera.
