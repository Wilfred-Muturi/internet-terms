Transmission control protocol / Internet Protocol

This is the fundamental communication language of the internet.

It is a set of rules that govern how data is broken to small packet addresses, transmitted and re-assemble at destination.

  • TCP - Breaks data into packets and ensure they arrive in order
  • IP - Handles the addressing for each packet

IP Address

A unique numerical address assigned to every machine on the internet.

The IPv4 address is a 32 bit binary number normally represented as 4 decimal values while the IPv6 addresses are written as eight 4-digit, hexadecimal numbers separated by colons, which represent 16 bits each.

An IP address and its subnet mask perform the following functions:

  • Enable the system to process the reciept and transmission of packets
  • They specify a range of addresses that share the cable with the device
  • They specify the devices local addresses
