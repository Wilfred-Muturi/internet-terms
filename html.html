Hyper text mark up language

HTML is platform independent in a way that web pages can be viewed on different devices and operating systems.

It describes the structure and content of a page e.g the headingd, paragraphs, images and links.

  • Works with CSS and JavaScript
  • Uses tags like <h1>, <p>, <a>

Web browsers read HTML code and display it as a formatted content for users.
