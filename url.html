Uniform Resource Locator (URL)

This is the complete, unique address of a resource on the web. It tells the browser how to get what from where.

A typical URL consists of several components, including the protocol, domain name and path to the resource.

Example: http://www.shule.com/index.html

Different parts of a URL

Part Example Explanation
Protocol https:// The set of rules for communication (e.g., HTTP or HTTPS).
Domain www.strathmore.edu The human-readable address of the server.
Port :443 The gate on the server. HTTP default is 80 while HTTPS is 443.
Path /academics/undergraduate The directory/folder structure on the web server loading the file.
Resource /courses.html The specific file or page being requested.
Query ?id=123&sort=asc Data sent to the server.
Fragment #section2 A bookmark to a specific part of the page.
