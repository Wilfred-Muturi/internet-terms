Hyper text transfer protocol

Hyper text transfer protocol (HTTP) is the communicationprotocol used on the World wide web to transfer data between web clients(browsers) and web servers.

It works on a request-response model. A web browser sends a request to a server and the server responds with the requested resource.

HHTP requests can be of different types including GET, POST, PUT, DELETE and others each serving a specific purpose.

  • Used for communication between browser and server
  • Transfers web pages and resources
  • Operates on request-response cycle
  • Foundation of web communication
