Domain Name System (DNS)

The domain name system is a system that translates human-readable domain names into machine-readable IP addresses that computers use to identify each other on the internet.

Since IP addresses are difficult to remember, DNS allows user to access websites using names such as www.google.com

Domain names are composed of different parts. They read from left to right.

  • At far right of the domain - you will find a TLD(top level domain). These are the .com, .org, .net
  • Middle level domain - The part after the www but before the top level domain
    • The final aspect is the www aspect - This is standard for every domain name and is referred to as the machine name.

    Domain name contains the following information:

    • The name of the organization
    • The organization itself
    • The country
    • The particular computer or network
