Web Servers

A program that stores web pages and serves content using HTTP. It listens for requests from browsers, retrieves the requested resource and send it back.

A web server can be both static content; pre-existing files such as HTML pages and images; and dynamic content; generated on demand based on user requests.

A web server:

  • Stores website files and resources
  • Serves static and dynamic content
  • Responds to HTTP requests
  • Supports web applications

A web server plays a critical role in enabling access to websites and web applications across the internet.
